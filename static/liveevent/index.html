Snail IDE Live Event

About the Event

A virus has made its way on the Snail IDE servers and it is up to the snails to destroy it and maybe have birthday cake on the way?

When Is The Event?

The event will be taking place at 7:00 PM EST on June 21st 2024 (depending on your time zone). Some time conversions are mentioned below.

  • PST - 4:00 PM, 6/21/24
  • MST - 5:00 PM, 6/21/24
  • CST - 6:00 PM, 6/21/24
  • CET - 1:00 AM, 6/22/24
  • EET - 2:00 AM, 6/22/24

By clicking the button below it will take you to the waiting room, you must be in there when the event starts and you can't be late. This is a multiplayer event so you will be able to see other snails. When the event is about to start join the waiting room and once it is time the event will begin.
